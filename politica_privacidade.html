Política de Privacidade

App: IF Estudantil – Bahia

Última atualização: 18 de agosto de 2025

1. Quem somos

Este aplicativo é um projeto independente e de código aberto, sem vínculo com o IFBA, SUAP ou qualquer órgão governamental. Ele atua como um cliente (wrapper) que consome a API pública do SUAP (Sistema Unificado de Administração Pública) para exibir, ao próprio usuário autenticado, suas informações acadêmicas.

2. Quais dados tratamos

  • Credenciais de login do SUAP (usuário e senha) — fornecidas pelo próprio usuário para autenticação no SUAP.
  • Dados acadêmicos retornados pelo SUAP (por exemplo: turmas, horários, notas, faltas, notícias).

3. Como usamos os dados

  • As credenciais são utilizadas exclusivamente para autenticar o usuário diretamente junto ao SUAP.
  • Os dados acadêmicos são exibidos no aplicativo apenas ao próprio usuário autenticado.
  • Não utilizamos dados para publicidade, perfis ou venda a terceiros.

4. Armazenamento e retenção

  • Não mantemos servidores próprios para armazenar dados pessoais de usuários.
  • As credenciais podem ser mantidas localmente no dispositivo apenas se o usuário optar por “manter sessão”. Essa informação permanece criptografada/local e pode ser apagada a qualquer momento ao sair da conta ou limpar os dados do app.
  • Os dados permanecem sob a responsabilidade do SUAP. O aplicativo atua somente como interface de acesso.

5. Compartilhamento

Não compartilhamos, vendemos ou transferimos dados de usuários a terceiros. Toda comunicação ocorre diretamente entre o aplicativo e o SUAP por meio de conexões seguras (HTTPS).

6. Segurança

  • Tráfego protegido por HTTPS.
  • Boas práticas de desenvolvimento seguro e revisão comunitária do código (open-source).
  • Recomendamos que o usuário utilize senha forte e não compartilhe suas credenciais.

7. Crianças e adolescentes

O aplicativo destina-se a estudantes que possuem acesso ao SUAP. Caso o ordenamento aplicável exija consentimento dos pais ou responsáveis, o acesso deve ser realizado pelo responsável legal.

8. Direitos do usuário

  • Acessar, revisar e excluir dados locais do app (ex.: sair da conta / limpar dados do aplicativo).
  • Solicitar esclarecimentos pelo canal de contato abaixo.

9. Cookies e rastreamento

Não utilizamos cookies de terceiros, SDKs de anúncios ou tecnologias de rastreamento.

10. Terceiros

Além do SUAP, não integramos outros serviços que coletem dados pessoais. Qualquer futura integração será documentada nesta política.

11. Alterações desta política

Podemos atualizar esta política para refletir mudanças no app ou requisitos legais. A versão vigente estará sempre disponível neste endereço, com a data de atualização no topo.
